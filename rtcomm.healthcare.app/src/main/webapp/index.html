Liberty Rtcomm Healthcare

This is a real-time communications help desk sample application. It provides both a customer and an agent perspective. To try this out you will need someone to play the role of both the customer and the agent. You can try this solo by opening two different browsers windows. You can also test this with multiple customers and agents. Enjoy!
